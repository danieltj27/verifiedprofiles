Add a verification badge to user profiles on your forum.

## Features

1. Add a verified badge next to the username of specific users (a notification will be sent when this happens).
2. Verified users can choose if they want to hife their badge (this is permission based).
3. Verify groups which allow users to be automatically verified when they join the group.
4. Upload a custom verification badge to overwrite the default badge included in the extension.
5. Toggle whether users must be verified again if they change their email address or username.

## Install

1. Upload the extension files into the `ext/danieltj/verifiedprofiles` directory.
2. Log in to the Admin Control Panel and install the extension.
3. Visit the extension settings page in the Admin Control Panel under the *Users and Groups* tab. Configure the settings to your liking.
4. Manage the user permissions to decide which users can hide their badge when they become verified.
5. Manage any user or group to determine who should have their profile verified and badge added to their profile.
